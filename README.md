# Driver_Monitoring_Terminal-main
本项目是一个基于 Django 开发的危险驾驶行为监测与预警系统后端项目，主要面向车辆终端、驾驶员管理和运维人员，提供危险驾驶行为数据接入、实时告警处理、消息推送以及后台管理等功能。系统支持接收终端上传的图片及相关事件数据，并根据不同类型的驾驶风险行为（如近距离碰撞、危险驾驶、疲劳驾驶、身份识别异常等）生成预警信息，进而通过消息系统向相关用户进行通知。

项目采用 Django 多应用模块化架构，包含用户认证与权限管理、消息通知、终端与车辆管理、平台运维管理、文档管理及操作记录等模块，具备较好的扩展性和实际部署能力。该系统可用于智能交通、车辆安全监控和驾驶行为管理等场景，为危险驾驶行为的识别与预警提供了后端支撑。

驾驶人疲劳驾驶行为识别技术及车载监控后台终端

运行指令：python manage.py runserver
